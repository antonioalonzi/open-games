# Open Games

Wonderful gaming platform application.



## Development

The application is written using:
 * Java (Spring)
 * Javascript (React)



### Configure

 * Install JDK 1.8
 * Install maven
 * Install npm
 
       npm i webpack-cli -D



### Build & Run

    cd src/main/resources/npm
    npm install
    ./node_modules/.bin/webpack -d
    cd -

    mvn spring-boot:run
